共享农场

YLE中的农场除了系统默认生成原生农场外，用户间可以自由组合Famland形成新的共享农场，只要已经在基本田存入种子基金的Famland数量达到30，即可创建新的共享农场。不同的农场，享有不同的APY。

各个农场的APY高低，取决于各个农场的YL分流指数，所谓YL分流指数，就是YLE新增发的YL从总的蓄水池中，流向该农场的YL流速。

农场内的Famland共享值越高、Famland数量越多、农场所获得的YL分流指数越大，农场的APY越高，农场内每个Famland所产出的YL越多。

{{ ":star:" | emojify }}

分流指数

农场的YL分流指数=（农场总的共享值/农场Famland总数量）*流速系数

农场的YL流速=（该农场分流指数/全部农场分流指数之和）*总流速

{{ ":rocket:" | emojify }}

流速系数

原生农场的流速系数恒定为1

共享农场的流速系数起步为1，以后共享农场每增加30个Famland，流速系数增加0.0001。

{{ ":gear:" | emojify }}
